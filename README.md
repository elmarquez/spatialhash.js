Spatial Hash
============

Fast spatial indexing and search for three.js.


Dependencies
------------

Install all development dependencies:

    npm install


Building a Distribution
-----------------------

Show build commands:

    grunt

Build the library:

    grunt compile

Test the distribution:

    grunt test
